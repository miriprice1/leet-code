# Leet-coder App

### Setup Instructions:

Before activation, you must perform the following steps:

1. **Run Docker and k8s on your local machine:**
    ```bash
    docker-compose up -d
    ```

2. **Run the Server:**
    for windows:
   ```bash
    .\start-server.exe
    ```
    for linux / macOS:
   ```bash
    ./start-server
    ```

4. **Run the App:**
    for windows:
   ```bash
    .\come-on-leet-code.exe
    ```
    for linux / macOS:
   ```bash
    ./come-on-leet-code
    ```
