# Leet-coder App

### Setup Instructions:

Before activation, you must perform the following steps:

1. **Run Docker Desktop on your local machine:**
    ```bash
    cd .DB/ && docker-compose up -d
    ```

2. **Run the Server:**
    ```bash
    cd ../SERVER/ && go run .
    ```

3. **Run the App:**
    ```bash
    cd ../UI/ && go run .
    ```
